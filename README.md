# 🧮 Counter App

A simple and interactive Counter App built with **HTML, CSS, and JavaScript**.

## ✨ Features

- ➕ Increase the counter
- ➖ Decrease the counter
- 🔄 Reset the counter to zero
- 🎨 Clean and responsive user interface

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
